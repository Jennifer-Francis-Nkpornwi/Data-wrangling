# Datawrangling Personal Project with SQL (MYSQL)

This is a project in which I remediated Significant Earthquake dataset from 1965 to 2016. 
This dataset was dowloaded from Kaggle and has 23412 rows and 21 columns

## Project Overview

This project demonstrates my ability to clean and prepare raw datasets using SQL. The goal was to transform unstructured and inconsistent data into a reliable format suitable for analysis and decision-making.

### Key Tasks Performed
Removed duplicate records

Handled missing/null values

Standardized data formats

Cleaned inconsistent entries

Improved overall data quality

### Tools Used
SQL (for data cleaning and transformation)
